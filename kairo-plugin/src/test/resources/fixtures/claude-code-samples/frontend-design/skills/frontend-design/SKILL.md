This skill guides creation of distinctive, production-grade frontend interfaces.

## Design Thinking

Before coding, understand the context and commit to a BOLD aesthetic direction.

## Frontend Aesthetics Guidelines

Focus on typography, color, motion, spatial composition, and visual details.

NEVER use generic AI-generated aesthetics like overused fonts (Inter, Roboto, Arial) or cliched color schemes.
