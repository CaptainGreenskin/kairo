# Logo Guidelines

Place your project logo here as `logo.svg` or `logo.png`.

Recommended specs:
- SVG format preferred (scalable)
- Square aspect ratio, 200x200px minimum
- Transparent background
- Works on both light and dark backgrounds

Once created, uncomment the logo line in README.md and README_zh.md:
```html
<p align="center"><img src="docs/logo.svg" alt="Kairo Logo" width="200" /></p>
```
