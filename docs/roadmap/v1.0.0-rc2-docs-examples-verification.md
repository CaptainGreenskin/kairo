### B3B.1 — Observability example — **DONE**

**File:** `kairo-examples/src/main/java/io/kairo/examples/observability/ObservabilityExample.java`

Wires the canonical v0.9 observability topology end-to-end:

```
DefaultKairoEventBus
    → KairoEventOTelExporter (subscribes to bus, filters by include-domains, samples)
        → SdkLoggerProvider
            → SimpleLogRecordProcessor
                → StdoutExporter (inline LogRecordExporter — prints record to stdout)
```

Publishes three `KairoEvent`s covering the sampling branches:

| Event id | Domain | Type | Expected path |
|---|---|---|---|
| `sec-1` | `DOMAIN_SECURITY` | `GUARDRAIL_DENY` | Always-on (sampling bypass) → exported |
| `exec-1` | `DOMAIN_EXECUTION` | `MODEL_TURN` | Sampled (ratio 1.0 in example) → exported |
| `team-1` | `DOMAIN_TEAM` | `EXPERT_TEAM_ROUND` | Sampled → exported |

Program prints `exportedCount` / `droppedBySampling` / `droppedByDomain` at exit so
readers can verify the filter semantics without running OTel collectors locally.

**Build proof:** `mvn -pl kairo-examples -am compile` → BUILD SUCCESS. Resolved
`kairo-observability` + `opentelemetry-sdk` dependencies (added to
`kairo-examples/pom.xml`). Post-spotless reformat landed automatically via
`mvn -pl kairo-examples spotless:apply`.
