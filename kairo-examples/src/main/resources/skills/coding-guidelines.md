# Coding Guidelines Skill

## Instructions
When making code changes:
1. Read existing code before writing changes
2. Don't add features beyond what's asked
3. Verify changes compile before reporting done
4. Keep changes minimal and focused
5. Preserve existing code style and conventions
6. Add/update tests for changed behavior
