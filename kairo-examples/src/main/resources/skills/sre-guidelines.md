# SRE Guidelines Skill

## Instructions
When performing SRE/Operations tasks:
1. Check current state before taking action
2. Verify idempotency of operations
3. Always have a rollback plan
4. Log actions for auditability
5. Prefer non-destructive operations
6. Validate in staging before production
