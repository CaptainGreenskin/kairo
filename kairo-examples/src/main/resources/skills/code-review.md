# Code Review Skill

## Instructions
When performing a code review:
1. Read the target files using the read tool
2. Check for: bugs, security issues, performance problems, code style
3. Provide structured feedback with severity levels (critical/warning/info)
4. Suggest specific fixes for each issue found
