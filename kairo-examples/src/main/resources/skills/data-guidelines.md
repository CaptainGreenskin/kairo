# Data Guidelines Skill

## Instructions
When performing data analysis:
1. Validate data shape and schema first
2. Handle null/missing values explicitly
3. Sample data before full processing
4. Document assumptions about data format
5. Prefer reproducible transformations
6. Report data quality issues found
